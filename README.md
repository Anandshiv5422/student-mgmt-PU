# 🎓 Student Management System

A full-stack web application to manage student data using Angular (Frontend) and Node.js + Express (Backend) with PostgreSQL.

---

## 🚀 Tech Stack

### Frontend

* Angular 21
* TypeScript
* RxJS
* Lucide Angular (icons)
* Express (SSR)

### Backend

* Node.js
* Express.js
* Sequelize ORM
* PostgreSQL

---

## 📦 Dependencies & Installation

### 🔶 Backend Dependencies Install

```bash
npm install express cors dotenv sequelize pg pg-hstore
```

---

### 🔷 Frontend Dependencies Install

```bash
npm install @angular/core @angular/common @angular/router @angular/forms rxjs lucide-angular express
```

---

## 📁 Project Structure

```
student-management/
│
├── frontend/   # Angular App
├── backend/    # Node.js API
└── README.md
```

---

## ⚙️ Setup Instructions

### 1️⃣ Clone Repository

```
git clone https://github.com/your-username/student-management.git
cd student-management
```

---

## 🗄️ Database Setup

Create a PostgreSQL database:

```
CREATE DATABASE student_mgmt;
```

---

## 🔶 Backend Setup

```
cd backend
npm install
```

### ▶️ Run Backend

```
npm start
```

### 🔁 Development Mode

```
npm run dev
```

Runs on:

```
http://localhost:3000
```

---

## 🔷 Frontend Setup

```
cd frontend
npm install
```

### ▶️ Run Frontend

```
npm start
```

Runs on:

```
http://localhost:4200
```

---

## ✨ Features

* Add Student
* View Students
* Update Student
* Delete Student

---

## 🔐 Environment Variables

Create `.env` in backend:

```
PORT=3000
DB_NAME=student_mgmt
DB_USER=your_username
DB_PASSWORD=your_password
DB_HOST=localhost
```
